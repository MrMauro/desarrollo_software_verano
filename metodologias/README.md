# Metodologías de Desarrollo de Software

Las metodologías de desarrollo de software son un conjunto de prácticas, procesos y reglas que guían cómo se planifica, desarrolla, prueba y mantiene un sistema de software.

Su objetivo principal es organizar el trabajo, reducir errores y mejorar la calidad del producto final.

## Clasificación

Las metodologías se pueden dividir en dos grandes grupos:

### Metodologías tradicionales

- Planificación rígida
- Fases bien definidas
- Poca flexibilidad ante cambios
- Ejemplo: Waterfall

### Metodologías Ágiles

- Desarrollo incremental
- Adaptación al cambio
- Entregas frecuentes
- Trabajo colaborativo
- Ejemplos: Scrum, Kanban

## ¿Por qué existen?

- Para ordenar equipos de trabajo
- Para controlar tiempos y costos
- Para reducir riesgos en proyectos grandes
- Para mejorar la comunicación
